# RedPackage-and-EatWhat-and-Rain
* ### About RedPackage
仿微信拼手气红包效果:  	
实现靠后方差大的随机数效果，参考[知乎链接](https://www.zhihu.com/question/22625187);		
红包总金额限定200元，并有限定设置;		
各种非空判定，每人最低抢到0.01元限定;

* ### About EatWhat
随机出已存入的餐厅来决定今天吃什么:  	
调用Random.org的API（对于小量数据的随机而言并没有什么卵用～）;  	
选用百度地图API，实现餐厅位置查找;
* ### About Rain
仿微信撒花效果:  	
透明界面;  	
撒落图片可选;
